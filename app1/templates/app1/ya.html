Сайт "Многоточие памяти"

Сайт "Многоточие памяти" был создан с использованием следующих инструментов: Python, JavaScript, Django, PostgreSQL, HTML, CSS, Ubuntu(Linux). Для развертывания использовался Docker. Для создания карты применялся PostgreSQL и Django REST framework(Python), а также JavaScript. Для хранения фотографий S3 хранилище на основе YandexCloud. Применяются API технологии для получения данных с других сайтов и вспомогательных страниц(погода и д.р.).

Сайт будет развиваться, а его отделы активно дополняться новыми данными.
